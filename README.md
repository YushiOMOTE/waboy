# waboy

Rust-generated wasm gameboy emulator.

* Based on [the no-std gameboy emulator](https://github.com/yushiomote/rgy).
* The demo is [here](http://139.180.193.221/).

## Key mapping

| Joypad | Keyboard |
|--------|----------|
| Right  | Right    |
| Left   | Left     |
| Up     | Up       |
| Down   | Down     |
| A      | Z        |
| B      | X        |
| Start  | N        |
| Select | M        |
